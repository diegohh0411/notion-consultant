# Tools for Notion Solution Partners

Hey, I'm Diego & I'm part of Notion's program for Solution Partners. That means I talk to people about Notion a lot, and to be a successful consultant I need to be able to articulate its features accurately & swiftly. Because the team at Notion is awesome and they ship so frequently, the docs change often and it can be difficult to keep up. So I've created these tools to help me stay up-to-date with the docs. 

The first of the tools is a scraper that reads Notion.com/help's XML sitemap and downloads all of the pages into markdown files. Then, we can feed these files to AI tools like NotebookLM to generate quizzes & summaries to help us keep in touch with the latest version of the docs.
